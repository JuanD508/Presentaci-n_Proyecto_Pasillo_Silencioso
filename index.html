Medidor de Gritos con ESP32

Un proyecto para mejorar la convivencia escolar

Introducción

En algunos colegios, los pasillos pueden generar mucho ruido por los gritos de los estudiantes, lo que interrumpe las clases y dificulta el aprendizaje. Este proyecto propone una solución tecnológica con ESP32, sensores y retroalimentación visual/auditiva para controlar el ruido.

Componentes utilizados

  • ESP32 Dev Kit
  • Sensor de micrófono (nivel de sonido)
  • Sensor PIR (detección de movimiento)
  • Pantalla LCD 16x2 con I2C
  • 3 LEDs (rojo, amarillo y verde) con resistencias de 220Ω
  • Módulo ISD1820 + altavoz
  • Protoboard y cables Dupont

Funcionamiento

LED Verde → ruido bajo.

LED Amarillo → ruido moderado.

LED Rojo → ruido alto (grito). Activa el ISD1820.

El PIR detecta movimiento y lo muestra en la LCD.

El ESP32 cuenta los gritos y los guarda en una página web.

Resultados

El sistema permite identificar en tiempo real el nivel de ruido y contabilizar los gritos diarios. Esta información se muestra tanto en la pantalla LCD como en una página web accesible por WiFi.

Conclusión

El proyecto combina IoT, sensores y señales visuales/auditivas para mejorar la convivencia en espacios escolares, fomentando ambientes más tranquilos y propicios para el aprendizaje.
